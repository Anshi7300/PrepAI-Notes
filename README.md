

  <h1 align="center">📚 PrepAI Notes Generator</h1>

  <p align="center">
    AI-powered notes generator to help students quickly create structured, exam-ready notes.
  </p>

  <h2>✨ Features</h2>
  <ul>
    <li>🧠 AI-generated notes from any topic</li>
    <li>📄 Clean and structured output (headings, bullet points)</li>
    <li>⚡ Fast and easy-to-use interface</li>
    <li>🔐 User authentication system</li>
    <li>📂 Save and manage your notes</li>
  </ul>

  <hr>

  <h2>🛠️ Tech Stack</h2>
  <ul>
    <li><b>Frontend:</b> React, Tailwind CSS</li>
    <li><b>Backend:</b> Node.js, Express.js</li>
    <li><b>Database:</b> MongoDB</li>
    <li><b>Authentication:</b> JWT</li>
    <li><b>AI Integration:</b> OpenAI API</li>
    <li><b>Deployment:</b> Render</li>
  </ul>

  <hr>

  <h2>🚀 How It Works</h2>
  <ol>
    <li>User enters a topic or question</li>
    <li>AI processes the input</li>
    <li>Structured notes are generated instantly</li>
    <li>User can save and access notes later</li>
  </ol>

  <hr>

  <h2>📦 Installation</h2>

  <h3>1. Clone the repository</h3>
  <pre>
git clone https://github.com/Anshi7300/PrepAI-Notes.git
cd PrepAI-Notes
  </pre>

  <h3>2. Install dependencies</h3>
  <pre>
npm install
  </pre>

  <h3>3. Setup environment variables</h3>
  <pre>
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
OPENAI_API_KEY=your_api_key
  </pre>

  <h3>4. Run the project</h3>
  <pre>
npm run dev
  </pre>
